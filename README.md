## Installation
```bash
git clone https://github.com/<dein-user>/my-neural-net.git
cd my-neural-net

```

## Nutzung
```bash
python main.py
```

- Trainiert ein MLP
- Speichert bestes Modell als `mlp_model.pt`
- Visualisiert die Entscheidungsgrenze
